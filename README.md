# Inflation Dashboard (Streamlit)

## Objective
To analyze and forecast inflation trends in essential commodities using time series methods.

## Features
- Time series visualization
- Moving averages
- Seasonal analysis
- Correlation heatmap
- Forecasting (Random Forest)

## Tech Stack
Python, Streamlit, Pandas, NumPy, Plotly, Scikit-learn

## How to Run
streamlit run app.py
